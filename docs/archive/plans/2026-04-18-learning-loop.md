# Learning Loop Revamp ‚Äî 2026-04-18

## Goal
Personal SWE growth tool. Every session = measurable improvement. Playground stays nucleus; everything else collapses into one tight loop.

## Core Loop
Today (one card) ‚Üí Playground (build + AI companion + ambient docs) ‚Üí Feynman gate (forced explain-back) ‚Üí Concept graph updates ‚Üí tomorrow's card smarter ‚Üí weekly brutal review.

## Data Preservation
- Existing `user_progress` (problem-level SM-2): kept verbatim, Anki keeps working.
- Existing problem JSONs (DSA/LLD/HLD/Behavioral): seed concept‚Üíproblem mapping.
- Existing library JSONs: seed concept‚Üísection mapping.
- Existing chats/notes: kept; surfaced ambient when concept matches.

## New Tables (additive, no migration of old data)
- `activity_log` ‚Äî every session event (concept_ids, duration, action)
- `concept_mastery` ‚Äî per-user per-concept FSRS state
- `daily_plan` ‚Äî cached today recommendation
- `weekly_review` ‚Äî Sunday AI report blob
- `feynman_logs` ‚Äî explanations + grades + extracted gaps

## New Endpoints
- `POST/GET /api/activity` ‚Äî log + range query
- `GET/POST /api/concepts` ‚Äî mastery map / FSRS review
- `POST /api/feynman` ‚Äî submit explanation, AI grades
- `GET/POST /api/daily` ‚Äî today's plan
- `GET/POST /api/weekly` ‚Äî weekly review
- `POST /api/tag` ‚Äî concept tagger (post-session AI)

## New Pages

- `/review` ‚Äî weekly report
